## 第一步：进化（独立运行）

```bash
cd 1-evolution/code
bash run_evolution.sh
```

步骤：分析(analyze) → 生成3个候选(generate) → 创建代码(create) → 验证(validate) → 跑10题评估 → Pareto选最优 → 下一轮

## 第二步：评测（独立运行）

```bash
cd 2-benchmark/code
bash run_xbench.sh voyager
bash run_xbench.sh cerebra_fusion_memory
bash run_xbench.sh temporal_experience_graph
# ... 支持 6 个 provider
```

步骤：加载 provider → 顺序跑 20 题 → 每步存取记忆 → judge 判分
